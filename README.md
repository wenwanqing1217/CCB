# 校园盲盒即时配送平台

基于微信小程序与云开发平台的校园闲置物品交易与即时配送系统。

## 项目简介

采用"盲盒交易 + 即时配送 + 社区互动"一体化设计，为高校学生提供便捷的闲置物品流转服务。

## 核心功能

### 盲盒交易
- 盲盒发布与购买
- 分类浏览与搜索
- 智能推荐

### 即时配送
- 骑手抢单机制
- 顺路匹配算法
- 实时配送追踪

### 社区互动
- 动态分享
- 即时通讯
- 积分系统

## 核心算法

### 顺路匹配算法
基于曼哈顿距离的骑手-订单动态匹配算法，综合考虑：
- 配送距离
- 骑手负载
- 时效要求

### 智能推荐算法
基于用户行为的协同过滤推荐：
- 浏览历史分析
- 分类偏好提取
- 个性化推荐生成
