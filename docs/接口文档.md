## 一、接口说明

### 1.1 基础信息

| 项目 | 说明 |
|------|------|
| 基础URL | `http://localhost:端口号/api` |
| 数据格式 | JSON |
| 字符编码 | UTF-8 |
| 鉴权方式 | JWT Token（Bearer Token） |

### 1.2 统一响应格式

```json
{
  "success": true,
  "data": { ... },
  "error": null
}
```

```json
{
  "success": false,
  "data": null,
  "error": "错误信息"
}
```

### 1.3 错误码

| 错误码 | 说明 |
|--------|------|
| 200 | 成功 |
| 400 | 参数错误 |
| 401 | 未登录/Token无效 |
| 403 | 无权限 |
| 404 | 资源不存在 |
| 500 | 服务器错误 |

## 二、user-service (端口: 8001)

### 2.1 登录/注册

**接口地址**：`POST /api/users/login`

**请求参数**：

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| openid | string | 是 | 微信OpenID |
| nickname | string | 否 | 用户昵称 |
| avatar | string | 否 | 头像URL |

**返回字段**：

| 字段 | 类型 | 说明 |
|------|------|------|
| token | string | JWT Token |
| userId | number | 用户ID |
| openid | string | 微信OpenID |
| nickname | string | 昵称 |
| avatar | string | 头像 |
| role | string | 角色 |
| status | string | 状态 |

**请求示例**：
```bash
curl -X POST "http://localhost:8001/api/users/login" \
  -H "Content-Type: application/json" \
  -d '{"openid": "oXXXX", "nickname": "张三"}'
```

**响应示例**：
```json
{
  "success": true,
  "data": {
    "token": "eyJhbGciOiJIUzI1NiJ9...",
    "userId": 1,
    "openid": "oXXXX",
    "nickname": "张三",
    "avatar": "https://...",
    "role": "BUYER",
    "status": "ACTIVE"
  }
}
```

## 九、错误码详解

| 错误码 | HTTP状态码 | 说明 | 解决方案 |
|--------|-----------|------|---------|
| TOKEN_EXPIRED | 401 | Token过期 | 重新登录 |
| TOKEN_INVALID | 401 | Token无效 | 重新登录 |
| USER_BANNED | 403 | 用户被禁用 | 联系管理员 |
| PERMISSION_DENIED | 403 | 无权限 | 检查角色权限 |
| RESOURCE_NOT_FOUND | 404 | 资源不存在 | 检查ID是否正确 |
| STOCK_NOT_ENOUGH | 400 | 库存不足 | 提示用户商品已售罄 |
| ORDER_GRABBED | 400 | 订单已被抢 | 提示用户订单已被其他骑手抢 |
| VALIDATION_ERROR | 400 | 参数校验失败 | 检查请求参数 |
