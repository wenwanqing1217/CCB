## 五、代码示例

### 生产者（Java）

```java
@Service
public class OrderService {
    
    @Autowired
    private RocketMQTemplate rocketMQTemplate;
    
    public Order createOrder(OrderCreateRequest request) {
        // 1. 创建订单
        Order order = orderDao.create(request);
        
        // 2. 构建消息
        OrderCreatedMessage message = new OrderCreatedMessage();
        message.setOrderId(order.getId());
        message.setBoxId(request.getBoxId());
        message.setBuyerId(request.getBuyerId());
        // ... 其他字段
        
        // 3. 发送消息
        rocketMQTemplate.convertAndSend("ORDER_CREATED", message);
        
        return order;
    }
}
```

### 消费者（Java）

```java
@Service
@RocketMQMessageListener(topic = "ORDER_CREATED", consumerGroup = "order-created-consumer")
public class OrderCreatedConsumer implements RocketMQListener<OrderCreatedMessage> {
    
    @Autowired
    private MessageService messageService;
    
    @Override
    public void onMessage(OrderCreatedMessage message) {
        log.info("收到订单创建消息: {}", message.getOrderId());
        
        // 推送订阅消息
        messageService.pushOrderCreatedNotification(message.getBuyerId(), message.getOrderId());
        
        // 发送站内信
        messageService.sendInboxMessage(message.getBuyerId(), "您的订单已创建");
    }
}
```

### 消费者（Go - delivery-service）

```go
type DeliveryService struct {
    producer *rocketmq.Producer
    consumer *rocketmq.PushConsumer
}

func (s *DeliveryService) StartConsumer() error {
    consumer, err := rocketmq.NewPushConsumer(
        consumer.WithGroupName("delivery-consumer"),
        consumer.WithNameServer([]string{"127.0.0.1:9876"}),
    )
    if err != nil {
        return err
    }
    
    err = consumer.Subscribe("ORDER_CREATED", consumer.MessageSelector{}, 
        func(ctx context.Context, msgs ...*primitive.MessageExt) (consumer.ConsumeResult, error) {
            for _, msg := range msgs {
                var orderMsg OrderCreatedMessage
                err := json.Unmarshal(msg.Body, &orderMsg)
                if err != nil {
                    log.Printf("解析消息失败: %v", err)
                    continue
                }
                
                // 查找附近骑手
                s.findNearbyRiders(orderMsg)
            }
            return consumer.ConsumeSuccess, nil
        })
    
    if err != nil {
        return err
    }
    
    return consumer.Start()
}
```

## 六、消息可靠性保证

### 1. 消息不丢

| 阶段 | 保证方式 |
|------|----------|
| 发送端 | 同步发送 + 重试 |
| 服务端 | 同步刷盘 + 多副本 |
| 消费端 | ACK 机制 + 重试 |

### 2. 消息不重复

使用幂等设计，每个消息有唯一ID：

```java
@Service
public class OrderCreatedConsumer implements RocketMQListener<OrderCreatedMessage> {
    
    @Autowired
    private RedisTemplate<String, String> redisTemplate;
    
    @Override
    public void onMessage(OrderCreatedMessage message) {
        // 1. 检查是否已处理
        String key = "msg:processed:" + message.getOrderId();
        if (redisTemplate.hasKey(key)) {
            return; // 已处理，直接返回
        }
        
        // 2. 处理业务
        processMessage(message);
        
        // 3. 标记已处理
        redisTemplate.opsForValue().set(key, "1", 1, TimeUnit.DAYS);
    }
}
```

### 3. 消息有序

使用相同 MessageKey 保证同一订单的消息顺序：

```java
rocketMQTemplate.syncSend("ORDER_STATUS_CHANGED", message, 
    new SendCallback() {}, 
    3000, message.getOrderId()); // orderId 作为 MessageKey
```

## 七、事务消息

### 场景：订单创建 + 扣库存

```java
// 1. 发送半事务消息
TransactionSendResult result = rocketMQTemplate.sendMessageInTransaction(
    "ORDER_CREATED_TX",
    message,
    orderId // 事务ID
);

// 2. 执行本地事务
try {
    orderDao.create(order);
    boxDao.deductStock(order.getBoxId());
    // 本地事务成功
    return LocalTransactionState.COMMIT_MESSAGE;
} catch (Exception e) {
    // 本地事务失败
    return LocalTransactionState.ROLLBACK_MESSAGE;
}

// 3. 事务回查（如果长时间没收到 commit/rollback）
@Override
public LocalTransactionState checkLocalTransaction(MessageExt msg) {
    String orderId = msg.getTransactionId();
    Order order = orderDao.findById(orderId);
    if (order != null) {
        return LocalTransactionState.COMMIT_MESSAGE;
    }
    return LocalTransactionState.ROLLBACK_MESSAGE;
}
```
