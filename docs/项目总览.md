# 校园盲盒即时配送平台 - 完整项目总览

## 项目简介

这是一个基于**微信小程序云开发**快速验证，并且已经设计并实现了完整**微服务架构升级**方案的校园盲盒即时配送平台，支持**C2C（个人对个人）**和**B2C（商家对个人）**双模式。

### 两种商业模式
| 模式 | 说明 | 场景示例 |
|------|------|---------|
| **C2C** | 个人发布闲置物品盲盒 | 学生出售二手书籍、电子产品等 |
| **B2C** | 认证商家发布商品 | 校园周边奶茶店、文具店、水果店等 |

## 核心功能

### 1. 🛍️ 买家端
- 盲盒浏览、搜索、分类筛选
- 热门推荐 + 个性化混合推荐
- 下单购买、订单管理
- 骑手位置实时查看、预估送达时间
- 积分系统、收藏管理

### 2. 🏪 卖家端（C2C + B2C）
- **个人卖家（C2C）**：
  - 发布闲置盲盒
  - 商品编辑、下架
  - 库存管理
  - 订单查看
- **认证商家（B2C）**：
  - 商家入驻申请
  - 批量发布商品
  - 库存管理
  - 订单查看、销售统计

### 3. 🏃 骑手端
- 附近订单查看（距离、收益）
- 抢单接单
- 配送状态更新（已取件/已送达）
- 位置实时上报
- 配送统计、收益管理

### 4. 🛠️ 管理端
- 用户管理（禁用/解禁）
- 商品审核、违规下架
- 订单监控、异常处理
- 商家入驻审核
- 数据统计（用户增长、订单量、交易额等）

## 架构说明

### 架构1：微信小程序云开发（快速验证）
- 技术栈：微信小程序 + 云开发（云函数/云数据库/云存储）
- 状态：**已完整实现**，可直接运行
- 特点：快速部署、开发成本低

### 架构2：微服务架构（大规模商业化）
- 技术栈：
  - Spring Boot（box-service、order-service、message-service）
  - Go（delivery-service）
  - Python（recommendation-service）
  - MySQL（主从复制）
  - Redis（集群 + Geo）
  - RocketMQ（消息队列）
  - Elasticsearch（搜索推荐）
  - Prometheus + Grafana + ELK + SkyWalking（监控运维）
- 状态：**已完成核心服务实现**
- 特点：高可用、高并发、易扩展

## API 总览

### box-service（商家端）
| 方法 | API | 说明 |
|------|-----|------|
| POST | /api/boxes | 发布盲盒 |
| PUT | /api/boxes/{boxId} | 更新盲盒 |
| PUT | /api/boxes/{boxId}/off | 下架盲盒 |
| GET | /api/boxes/{boxId} | 盲盒详情 |
| GET | /api/boxes/seller/{sellerId} | 商家商品列表 |
| GET | /api/boxes | 首页商品列表 |
| POST | /api/boxes/{boxId}/deduct | 扣库存（内部） |
| POST | /api/boxes/{boxId}/restore | 恢复库存（内部） |

### order-service
| 方法 | API | 说明 |
|------|-----|------|

| GET | /api/orders/{orderId} | 订单详情 |
| PUT | /api/orders/{orderId}/status | 更新订单状态 |

### delivery-service
| 方法 | API | 说明 |
|------|-----|------|

### recommendation-service

## 技术亮点

1. **混合推荐算法**：内容推荐 + 协同过滤，60:40权重混合
2. **原子性并发控制**：抢单使用原子更新，防止超卖
3. **完整RBAC权限体系**：用户/骑手/管理员三级权限
4. **Redis Geo**：实现附近骑手和订单查询
5. **消息队列解耦**：RocketMQ异步处理订单和消息推送
6. **多语言微服务**：Spring Boot + Go + Python，发挥各语言优势
7. **完整监控体系**：Prometheus + Grafana + ELK + SkyWalking
8. **分库分表设计**：应对数据量增长
9. **完整测试体系**：50+测试用例、接口自动化、AI专项评测、并发测试
10. **AI集成**：豆包API接入、智能客服、个性化推荐
